# PrismaX Brand Studio

Professional high-resolution content maker for digital branding. Built with Fabric.js and Express.

## Features
- Dynamic Asset Library (Logos, Stickers, Backgrounds)
- Master-Scale PNG Export (Full HD)
- Advanced Layer Management
- Custom Gradient Designer
- Professional Layout Blocks

## Deployment
This project is configured for Vercel deployment. Ensure you have the `vercel.json` file in the root.

## Local Development
1. `npm install`
2. `npm start`
3. Open `http://localhost:5000`
